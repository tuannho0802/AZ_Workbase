Tính năng nổi bật

API được thiết kế để mở rộng, bảo mật và dễ sử dụng

Bảo mật JWT

Xác thực bằng JWT với refresh token, bảo vệ API khỏi truy cập trái phép.

RBAC

Phân quyền theo vai trò (Admin, Manager, Employee) với kiểm soát truy cập chi tiết.

TypeORM + MySQL

Database được tối ưu với indexes, soft delete và migration tự động.

Performance

Server-side pagination, caching và query optimization cho 8K+ records.

Swagger Docs

Tài liệu API tự động với Swagger UI, hỗ trợ test trực tiếp.

Analytics

Thống kê real-time về khách hàng, giao dịch và hiệu suất sales.
